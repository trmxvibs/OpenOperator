# OpenOperator

OpenOperator is an open-source framework for building AI agents that can understand and operate computers like humans.

Observe. Reason. Act.

Unlike traditional automation tools that depend on APIs or predefined scripts, OpenOperator aims to interact with software through visual understanding, accessibility information, and direct mouse/keyboard control.

## Vision

Build an open ecosystem for computer-operating AI agents.

## Core Principles

* Cross Platform
* Open Source
* Modular Architecture
* Contributor Friendly
* Human-Like Computer Interaction
